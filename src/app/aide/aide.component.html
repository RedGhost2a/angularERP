Installation de Tailscale
Qu'est ce que Tailscale?

Tailscale est VPN qui est principalement un moyen de créer un réseau privé virtuel sur lequel peut se rejoindre l’ensemble de vos appareils (smartphones, ordinateurs, tablettes…etc.) pour communiquer et partager des fichiers au travers d’Internet en toute confidentialité, comme s’ils étaient sur un réseau local.

Dans notre cas, il créera un "tunnel" de connexion securisé entre vos ordinateurs et le serveur ou l'application d'études de prix est héberger.

Instation Tailscale?

Tailscale est disponible sur tout les principaux OS (systéme d'exploitation). Ce guide vous ménera pas à pas depuis l'instalation de cette application jusqu'à la connexion à votre application d'études de prix.

  • 1)Demander à votre administrateur l'ajout de votre profil nécessaire pour sécuriser votre connexion.
  • 2)Télécharger l'application ici.

    * Dans le cas ou vous auriez fournit votre adresse mail à votre administrateur, vous recevrez également un lien d'invitation dans votre boite mail.

  • 3)Installer le logiciel sur le support qui vous servira à vous connecter à l'application GRAZIANI Etude de prix.
  • 4)Connecter vous a l'application à l'aide de l'adresse mail que vous avez fourni à votre administrateur web.

    *Pour vous connecter vous devrez recevoir une confirmation par votre administateur de l'ajout de votre profil a Tailscale pour valider votre support de connexion.

 Connexion au site d'études de prix

Une fois l'installation Tailscale terminer. Ouvrez le logiciel, vous y verrez une adresse IP( ex: 100.85.96.112).

Il vous suffira ensuite de copier cette IP qui correspond à l'adresse de votre site et de rajouter le port 800. Vous aurez dans votre barre de votre navigateur une adresse comme suit: http://100.85.96.112:800/

Vous voila desormais connecter a votre application!
