Options d'administration

info

Bienvenue sur la page de gestion des entreprises. Cet espace vous permet d'accéder à différentes fonctionnalités associées à la gestion de votre entreprise : 1. Ajouter une entreprise : Cliquez sur le bouton add pour enregistrer une nouvelle entreprise dans le système.
2. Liste des devis : Cliquez sur l'icônelist pour accéder à la liste complète de tous les devis enregistrés.
3. Ajouter un utilisateur : Cliquez sur l'icônecreate pour ajouter un nouvel utilisateur à une entreprise.
4. Importer des données : Utilisez l'option import_export pour importer des données pertinentes pour votre entreprise à partir d'un fichier externe.
5. Créer une unité : Cliquez sur playlist_add pour créer et enregistrer une nouvelle unité pour votre entreprise.
Naviguez à travers ces options pour gérer efficacement les informations de votre entreprise.

Liste des tickets inter-entreprises

info

Vous vous trouvez actuellement sur l'onglet de Gestion des tickets. Cet espace vous permet de consulter et de gérer les tickets inter-entreprises. Voici un aperçu de ce que vous pouvez faire : 1. Consulter les tickets : Chaque ticket est présenté dans un panneau déroulant. Le titre du ticket, le type d'erreur, la date de création, et un badge indiquant la présence de logs sont affichés en tête de chaque panneau. 2. Consulter les détails du ticket : Cliquez sur un panneau pour le dérouler et consulter plus de détails sur le ticket, y compris le sujet et les messages de log associés. 3. Marquer un ticket comme résolu : Cochez la case 'Résolu !' pour indiquer qu'un ticket a été résolu. Cette action peut être inversée en décochant la case si nécessaire. 4. Supprimer un ticket : Si un ticket n'est plus pertinent ou nécessaire, vous pouvez le supprimer en cliquant sur le bouton 'Supprimer'. Utilisez ces fonctionnalités pour assurer un suivi efficace des tickets et maintenir une communication fluide entre les entreprises.
